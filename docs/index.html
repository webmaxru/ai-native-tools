Install

Get all plugins at once via the plugin marketplace

Plugin Marketplace

/plugin marketplace add webmaxru/ai-native-dev

Then install individual plugins:

/copilot plugin add ai-native-dev-skills@webmaxru/ai-native-dev /copilot plugin add web-ai-skills@webmaxru/ai-native-dev /copilot plugin add enonic-skills@webmaxru/ai-native-dev

APM (Individual Skills)

apm install webmaxru/ai-native-dev/skills/agent-package-manager apm install webmaxru/agent-skills/skills/prompt-api apm install webmaxru/enonic-agent-skills/skills/enonic-api-reference

Replace the skill path with any skill from the catalog above.
